# 🧵 Tailor App – Final Project Submission

## 📌 Project Description

Tailor App is a Flutter-based application designed to simplify tailoring business operations. It helps manage customer orders, measurements, and services in an efficient and user-friendly way.

This project is developed as a final submission showcasing frontend and backend integration.

---

## ✨ Features

### 👤 User Side

* User Registration & Login
* Place stitching orders
* Add and manage measurements
* View order status

### 🧵 Tailor/Admin Side

* Manage customer orders
* Track measurements
* Service management (stitching, alteration, etc.)
* Dashboard overview

## ⚙️ Setup Instructions

### 🔹 Frontend Setup

1. Clone the repository:
   git clone https://github.com/your-username/tailor_app.git

2. Navigate to project folder:
   cd tailor_app

3. Install dependencies:
   flutter pub get

4. Run the app:
   flutter run

### 🔹 Backend Setup (if included)

1. Go to backend folder:
   cd backend

2. Install dependencies:
   npm install

3. Start server:
   npm start
